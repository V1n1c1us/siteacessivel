Opções de Acessibilidade do Site

Botão para “Aumentar Letra” do conteúdo do site.

Botão para “Diminuir Letra” do conteúdo do site.

Botão para definir “Letra normal”, que faz com que o tamanho da letra, do conteúdo do site, volte ao tamanho padrão.

Botão para ativar o “Auto Contraste”, que define a cor de fundo do site para a cor preta e, a cor da letra do conteúdo para a cor branca.

Botão para ativar as “Cores originais”, que define a cor de fundo do site para a cor branca e, a cor da letra do conteúdo para a cor preta.

Áudio que é emitido automaticamente para informar “Página Carregada” para indicar a possibilidade de iniciar a interação com a página.
